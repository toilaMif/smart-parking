# Smart Parking Mobile

App mobile cho khach hang quet ma QR gui xe.

## Chay app

```powershell
cd "C:\Users\nguye\OneDrive\Máy tính\IOT_CK\smart-parking\mobile"
npm.cmd install
npm.cmd start
```

Sau do cai app Expo Go tren dien thoai va quet QR terminal hien ra.

## Chuc nang

- Xin quyen camera.
- Quet ma QR.
- Hien thi noi dung QR.
- Copy noi dung QR.
- Neu QR la link `http/https`, co the mo link truc tiep.
